# Idea del Proyecto: NetAdmin-Hub

## Descripción general

NetAdmin-Hub es una plataforma web "todo en uno" pensada para simplificar el trabajo diario de quienes gestionan redes e infraestructura informática. Centraliza herramientas de cálculo, guías de referencia rápida y un asistente inteligente que ayuda a resolver problemas técnicos sin tener que buscar en múltiples fuentes.

---

## Problema que intenta resolver

Los administradores de redes y técnicos de sistemas suelen necesitar varias herramientas distintas para su trabajo diario: calculadoras de subredes, tablas de referencia de puertos y protocolos, generadores de configuraciones, documentación técnica dispersa en diferentes webs... Esto supone perder tiempo navegando entre pestañas y sitios poco amigables.

NetAdmin-Hub nace para resolver eso: tener todo en un solo lugar, con una interfaz moderna, rápida y fácil de usar.

---

## Usuario objetivo

- Administradores de redes y sistemas en empresas.
- Técnicos de soporte e infraestructura IT.
- Estudiantes de redes y telecomunicaciones.
- Cualquier profesional técnico que trabaje con redes TCP/IP, switches, routers o servidores.

---

## Funcionalidades principales

1. **Calculadora de subredes (CIDR/VLSM):** Introduce una dirección IP y una máscara y obtén al instante la red, broadcast, rango de hosts, wildcard mask y número de hosts disponibles.
2. **Conversor de unidades de red:** Convierte entre Mbps, Kbps, Gbps, bytes por segundo, etc.
3. **Tabla de puertos y protocolos:** Consulta rápida de los puertos más comunes (HTTP, HTTPS, SSH, FTP, DNS, SMTP…) con descripción y protocolo asociado.
4. **Asistente inteligente (IA):** Chat integrado con IA para resolver dudas técnicas sobre redes, configuraciones, errores comunes o conceptos de networking.
5. **Guías de referencia:** Colección de cheatsheets y guías rápidas (modelo OSI, comandos de red por sistema operativo, subnetting, etc.).
6. **Panel de inicio:** Dashboard con acceso rápido a todas las herramientas.

---

## Funcionalidades opcionales

- Historial de cálculos guardado en LocalStorage.
- Modo oscuro / claro.
- Favoritos para guardar consultas frecuentes.
- Exportar resultados de calculadoras a PDF o texto.
- Buscador global de contenidos dentro de la app.

---

## Posibles mejoras futuras

- Autenticación de usuarios para guardar preferencias en la nube.
- Calculadora de VLANs y topologías de red.
- Integración con APIs de monitorización (ping, traceroute desde el servidor).
- Versión PWA (Progressive Web App) para uso offline.
- Soporte multiidioma (inglés / español).
- Sección de comunidad o foro técnico.
